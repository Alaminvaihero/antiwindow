# 🛡️ MR VIRUS Terminal Antivirus Scanner

![Banner](https://via.placeholder.com/800x200/0d1117/00ffcc?text=MR+VIRUS+Terminal+Antivirus+Scanner)

**MR VIRUS Terminal Antivirus Scanner** হলো একটি শক্তিশালী, ওপেন-সোর্স অ্যান্টিভাইরাস স্ক্যানার যা সম্পূর্ণরূপে টার্মিনাল থেকে চালানো যায়। এটি Python দিয়ে তৈরি এবং Windows 10/11-এর জন্য অপটিমাইজ করা হয়েছে। এটি **হ্যাশ ডাটাবেজ**, **YARA রুলস**, **Windows AMSI API**, এবং **PE ফাইল বিশ্লেষণ** ব্যবহার করে ম্যালওয়্যার সনাক্ত করতে সক্ষম।

## 📥 উইন্ডোজে কীভাবে ইন্সটল করবেন (বিগিনার-ফ্রেন্ডলি)

### 🐍 ধাপ ১: Python ইন্সটল করুন
1. [python.org](https://www.python.org/downloads/) ওয়েবসাইটে যান।
2. **Download Python** বাটনে ক্লিক করে সর্বশেষ সংস্করণটি ডাউনলোড করুন।
3. ডাউনলোড করা `.exe` ফাইলটি চালান।
4. **গুরুত্বপূর্ণ:** ইন্সটলার উইন্ডোতে **"Add Python to PATH"** চেকবক্সটি টিক করে দিন।
5. **Install Now** বাটনে ক্লিক করে ইন্সটল সম্পন্ন করুন।

### 📂 ধাপ ২: টুলটি ডাউনলোড করুন
আপনার পছন্দের একটি ফোল্ডারে (যেমন `C:\Users\YourName\Desktop`) টুলটি ডাউনলোড করুন। আপনি দুটি উপায়ে এটি করতে পারেন:

#### অপশন A: Git ব্যবহার করে (সহজ)
1. [Git for Windows](https://git-scm.com/download/win) ইন্সটল করুন।
2. Command Prompt খুলুন এবং নিচের কমান্ডটি রান করুন:
